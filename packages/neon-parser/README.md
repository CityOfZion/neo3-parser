# Neon-parser

## Install

```bash
npm i @cityofzion/neon-parser
```

## Using NeonParser
To use Neonparser you can simply import it and use the object methods.
```ts
import { NeonParser } from '@cityofzion/neon-parser'

NeonParser.formatResponse(response)
```
